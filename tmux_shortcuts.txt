==========================
===  TMUX  ===
==========================

[crtl + d] c = new window
[crtl + d] X = close current window
[crtl + d] x = Close current pane

shift + left = Previous window
shift + right = Next window
alt + arrows = Switch to pane to the direction

[crtl + d] h = Split pane horizontally
[crtl + d] v = Split pane vertically

[crtl + d] ! = Convert pane into a window

spacebar = select work
enter = copy
[crtl + d] p = paste
